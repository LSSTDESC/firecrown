firecrown.cosmology module¶

firecrown.cosmology.get_ccl_cosmology(input_params)¶

Get an input CCL cosmology using a cache.

Parameters:
input_params : dict

An input parameter dictionary. The name of the parameters should match the CCL name. Parameters not in the CCL namespace are ignored. The value can be a float or a list of three floats. In the case that a list is passed, the middle value is used.
Returns:
cosmo : pyccl.Cosmology

The CCL Cosmology object.
