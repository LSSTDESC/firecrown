class firecrown.ccl.systematics.lss.LinearBiasSystematic(alphaz, alphag, z_piv)¶

Bases: firecrown.ccl.core.Systematic

Linear bias systematic.

This systematic adds a linear bias model which varies with redshift and the growth function.

Parameters:
alphaz : str

The mame of redshift dependence parameter of the linear bias.

alphag : str

The name of the growth dependence parameter of the linear bias.

z_piv : str

The name of the pivot redshift parameter for the linear bias.

Magnification bias systematic.

This systematic adds a magnification bias model for galaxy number contrast following Joachimi & Bridle (2010), arXiv:0911.2454.

Parameters:

The name of the limiting magnitude in r band filter.

The name of the fitting parameters in Joachimi & Bridle (2010) equation (C.1).
